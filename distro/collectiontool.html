This Collection Tool is used to collect diagnostic information from the SAS App Engine. This includes log files, configuration information, disk usage data, database information, and hardware data. rPath Support may request this information when handling support issues related to this rBuilder appliance.

Click Collect Data to run the collection tool and collect all of the latest diagnostic information into a single archive saved on the SAS App Engine.

After running the collection tool, click Download Now to download the archive. Unpack and view the archive contents if desired. When filing a support issue in the rPath Issue Tracking System (issues.rpath.com), attach this archive to the issue.
