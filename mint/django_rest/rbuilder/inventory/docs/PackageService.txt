Package properties:
    created_by - the user that created the resource
    created_date - the date the package was created (UTC)
    description - Description of package
    modified_by - the user that last modified the resource
    modified_date - the date the package was last modified (UTC)
    name - Name of package
    package_id - Database id of package

Methods:
    GET:
        Authentication: user
    POST:
        Authentication: user
    PUT:
        Authentication: user
    DELETE:
        not supported
